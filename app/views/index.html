is an open-source load testing framework based on Scala, Akka and Netty
High performances
HTTP support
Simple concepts
Scenario recorder
A DSL to describe scenarios
Meaningful reports

Write code

val scn = scenario("Gatling")
  .group("Creation process") {
    exec {
      http("index").get("/")
    }.exec {
      http("new").get("/computers/new")
    }.pause(3.seconds).exec {
      http("create")
        .post("/computers")
        .param("name", "Gatling Stress Tool")
        .check(css("div.alert-message").exists)
    }
  }

Bootstrap it

val baseHttpConf = http
  .baseURL("http://localhost:9000")
  .disableAutoReferer

val httpConf = baseHttpConf
  .acceptEncodingHeader("")

setUp(
  scn.inject(
    atOnce(1.user)))
  .protocols(httpConf)
  .pauses(UniformPercentage(0.6))

Launch it

Inside your favorite terminal, with:
[G@ling bundle ]$ ./gatling.sh
or
[G@ling maven-project ]$ mvn gatling:execute
or
[G@ling sbt-project ]$ sbt gatling:test
